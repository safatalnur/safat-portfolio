# Alnur Theme Basic Skeleton

A very minimal lightweight and customizable WordPress theme designed for modern websites with Dart Sass included

## Description
This theme provides a clean skeleton layout built with WordPress best practices. It supports custom templates, theme options.

## Features 
- Customizable header, footer, and sidebar widgets  
- Theme Customizer support  
- Dart Sass included

## Installation
1. Download or clone this repository:
   ```bash
   git clone https://github.com/username/theme-name.git

2. For Dart Sass compilation, from the root directory, run
   ```bash
   sass --watch sass/style.scss assets/css/alnur-style.css
